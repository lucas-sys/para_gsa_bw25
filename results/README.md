# Results Directory

This directory stores all output files generated by the LCA GSA workflow.

## Directory Structure

```
results/
├── deterministic/              # Deterministic LCA scores
├── contribution/               # Contribution analysis results (Tier 2)
├── sensitivity_oat/            # One-at-a-time (OAT) sensitivity analysis
├── sensitivity_analytical/     # Analytical variance sensitivity analysis
├── monte_carlo/                # Monte Carlo simulation results & parameter samples
└── sobol/                      # Sobol global sensitivity analysis results
```

## Output Files

| Directory | File | Description |
|-----------|------|-------------|
| `deterministic/` | `deterministic_results.xlsx` | Baseline LCA scores for all FU/method pairs |
| `contribution/` | `contribution_analysis.xlsx` | Recursive contribution analysis (Tier 2) |
| `contribution/` | `formula_check_results.xlsx` | Exchange formula validation report |
| `sensitivity_oat/` | `oat_sensitivity_results.xlsx` | OAT finite-difference sensitivity coefficients |
| `sensitivity_analytical/` | `analytical_sensitivity_results.xlsx` | Analytical variance decomposition |
| `monte_carlo/` | `monte_carlo_results.xlsx` | Monte Carlo LCA scores per iteration |
| `monte_carlo/` | `parameter_samples.xlsx` | Sampled parameter matrix |
| `sobol/` | `sobol_results.xlsx` | First-order (S1) and total (ST) Sobol indices |
| `sobol/` | `sobol_raw_results.pkl` | Raw evaluation results (pickle) |
